### GetStarted
1. Клонировать репо
2. Перейти в папку проекта
3. `npm i` - установка всех зависимостей
4. `npm run dev` - поднять проект локально на 3000 порту в режиме разработки
- для работы нужна node.js
- порт можно изменить в vite.config.ts
